# NetDiag: Утилита для диагностики сети

Простая утилита для комплексной диагностики сетевых проблем.

---

## Способы запуска

### 1. Удаленный запуск (без установки)

Этот способ позволяет запустить программу без сохранения каких-либо файлов на вашем устройстве. Все будет загружено во временный каталог и автоматически удалено после завершения.

**Требования:** `bash`, `curl`, `tar`, `python`.

Выполните в терминале:
```bash
curl -sSL https://raw.githubusercontent.com/zootsman/netdiag/master/run.sh | bash
```

### 2. Локальная установка

Этот способ клонирует репозиторий на ваше устройство для постоянного использования.

**Требования:** `bash`, `curl`, `git`, `python`.

Выполните в терминале:
```bash
curl -sSL https://raw.githubusercontent.com/zootsman/netdiag/master/install.sh | bash
```
После установки следуйте инструкциям в терминале для запуска программы.

# NetDiag: A Network Diagnostic Utility

A simple utility for comprehensive diagnosis of network problems.

## How to Run

### 1. Remote Execution (No Installation)

This method allows you to run the program without saving any files to your device. Everything will be downloaded to a temporary directory and automatically deleted upon completion.

**Requirements:** `bash`, `curl`, `tar`, `python`.

Run in your terminal:
```bash
curl -sSL https://raw.githubusercontent.com/zootsman/netdiag/master/run.sh | bash
```

### 2. Local Installation

This method clones the repository to your device for permanent use.

**Requirements:** `bash`, `curl`, `git`, `python`.

Run in your terminal:
```bash
curl -sSL https://raw.githubusercontent.com/zootsman/netdiag/master/install.sh | bash
```
After installation, follow the instructions in the terminal to run the program.
